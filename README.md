# 🧠 OCD & Anxiety Detection System

## 📌 Overview
This project is a machine learning–based application that analyzes user inputs and detects patterns related to Obsessive-Compulsive Disorder (OCD) and anxiety levels. It aims to provide early insights into mental health conditions using data-driven techniques.

## 🎯 Objective
To develop a system that can identify possible OCD and anxiety symptoms based on behavioral and psychological data.

## ⚙️ Features
- Detects OCD and anxiety patterns  
- Uses machine learning algorithms  
- Simple and user-friendly interface  
- Fast and efficient predictions  

## 🛠️ Technologies Used
- Python  
- Machine Learning Algorithms  
- Pandas, NumPy  

## 🚀 How to Run
1. Clone the repository  
2. Install required libraries  
3. Run the Python file  

## ⚠️ Disclaimer
This project is for educational purposes only and does not replace professional medical diagnosis.
